# Terraform provider for Exoscale

This is a [Homebrew Tap][tap-doc] for the [Exoscale provider for
Terraform][exoscale-provider].

Install with:

    $ brew install --HEAD Mayeu/terraform-provider-exoscale/terraform-provider-exoscale

Enjoy.
